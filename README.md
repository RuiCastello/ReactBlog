# ReactBlog
### A simple blog website made in react:

- [DEMO](https://demos.canalfoto.org/blogapp) 

<br />



This app uses:  
- Class Components
- HoC's (higher order components)
- Lazy loading
- React Router
- State management
- Extensive use of lifecycles
- Axios
- etc.

Built while learning React in an online course
